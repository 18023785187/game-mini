# 项目说明

## 项目概述

这是一个2.5D横屏微信小游戏项目，类似拳皇对战的格斗游戏。

## 技术栈

- Node 22
- Pnpm 8
- TypeScript
- CloudBase (微信云开发)

## 项目结构

```
game-mini/
├── SPEC/                    # 需求文档目录
│   ├── Me2AI/              # 用户维护的需求迭代文档
│   └── AI2AI/              # AI维护的项目文档
├── src/                    # 源代码目录（工作区）
│   ├── config/             # 配置文件
│   │   ├── secrets.ts      # 敏感信息（不提交git）
│   │   └── secrets.example.ts
│   ├── game.ts             # 游戏主入口
│   └── game.json           # 游戏配置
├── .husky/                 # Git hooks
├── package.json
├── tsconfig.json
├── project.config.json     # 微信项目配置（不提交git）
└── project.config.example.json
```

## 开发指南

### 环境准备

1. 安装依赖：`pnpm install`
2. 配置敏感信息：
   - 复制 `src/config/secrets.example.ts` 为 `src/config/secrets.ts`
   - 填写 `APP_ID` 和 `CLOUD_BASE_ENV`
   - 复制 `project.config.example.json` 为 `project.config.json`
   - 填写 `appid`

### 开发命令

- `pnpm dev` - 开发模式，监听文件变化
- `pnpm build` - 构建项目
- `pnpm lint` - 代码检查
- `git cz` - 使用 commitizen 提交代码

## Git 管理

- 使用 husky + lint-staged + commitizen 管理
- 提交前自动执行 lint 检查
- 使用约定式提交规范

## 注意事项

1. 敏感信息文件不要提交到 git
2. 保持 example 文件与实际文件同步
3. 代码风格保持一致
