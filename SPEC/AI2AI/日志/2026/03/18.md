## 六、神枪手举枪动作与子弹发射修复

### 需求描述

用户需求：
- 神枪手还是没有抬枪设计，要抬到眼前一个水平线
- 子弹从枪口射击
- 现在看不到子弹射出

### 问题分析

**问题1：举枪角度不够高**
- 之前：举枪从-0.3弧度到0.3弧度（约17度到-17度）
- 问题：枪口没有抬到眼前水平线

**问题2：子弹发射位置不正确**
- 之前：子弹从 `character.y + 30` 发射
- 问题：位置不对，不在枪口位置

**问题3：子弹渲染坐标错误**
- 之前：在 `renderProjectiles` 中 translate 到屏幕坐标后，`renderBullet` 又 translate 了一次世界坐标
- 问题：双重偏移导致子弹不可见

### 功能实现

#### 1. 修复举枪角度 (`src/renderers/GunnerRenderer.ts`)

将举枪角度调整到眼前水平线：

```typescript
// 初始状态：手臂下垂，枪口向下（-1.2弧度≈-70度）
let leftGunAngle = -1.2;

if (isAttacking) {
  if (attackProgress < 0.3) {
    // 前半段：举枪动作（从下垂到眼前水平线）
    leftGunAngle = -1.2 + (attackProgress / 0.3) * 1.1; // 从-1.2到-0.1（几乎水平）
  } else {
    // 后半段：射击 + 后坐力
    const shootProgress = (attackProgress - 0.3) / 0.7;
    leftGunAngle = -0.1 + Math.sin(shootProgress * Math.PI * 3) * 0.2;
  }
}
```

**角度变化**：
- **初始**：-1.2弧度（-70度）- 手臂下垂，枪口向下
- **举枪后**：-0.1弧度（-6度）- 枪口几乎水平，对准前方

## 十六、炸弹抛物线运动

### 需求描述

用户需求：
- 炸弹应该是抛物线轨迹，而不是直线飞行

### 功能实现

#### 1. 抛物线运动变量 (`src/entities/Projectile.ts`)

```typescript
// 位置和移动
private startY: number;       // 起始Y坐标（用于计算抛物线）
private velocityY: number;    // Y方向速度（抛物线用）
private gravity: number = 600; // 重力加速度（像素/秒²）

// 炸弹抛物线专用
private targetX: number;      // 目标X坐标
private flightTime: number = 0; // 飞行时间
```

#### 2. 抛物线初速度计算 (`src/entities/Projectile.ts`)

```typescript
// 炸弹使用抛物线运动
if (config.type === ProjectileType.BOMB) {
  this.targetX = x + (direction === CharacterDirection.RIGHT ? config.range : -config.range);
  // 计算抛物线初速度
  // 假设炸弹飞行时间为1.5秒
  const flightDuration = 1.5;
  this.velocityX = (this.targetX - x) / flightDuration;
  // 计算初始Y速度，使炸弹能够达到最高点后落下
  const maxHeight = 150; // 最高点高度
  this.velocityY = (maxHeight + 0.5 * this.gravity * Math.pow(flightDuration / 2, 2)) / (flightDuration / 2);
}
```

**计算原理**：
- X方向：匀速运动，`velocityX = distance / time`
- Y方向：抛物线运动，初速度使炸弹上升到最高点后下落
- 飞行时间：1.5秒
- 最高点高度：150像素

#### 3. 抛物线更新 (`src/entities/Projectile.ts`)

```typescript
// 炸弹使用抛物线运动
if (this.config.type === ProjectileType.BOMB) {
  // Y方向：抛物线运动
  // y = y0 + v0 * t - 0.5 * g * t²
  this._y = this.startY + this.velocityY * this.flightTime - 0.5 * this.gravity * Math.pow(this.flightTime, 2);

  // 检查是否落地（Y坐标降到起始高度或以下）
  if (this._y <= this.startY && this.flightTime > 0.1) {
    this._y = this.startY;
    this.startExplosion();
  }
}
```

**运动方程**：
```
y(t) = y₀ + v₀·t - ½·g·t²
```

**落地检测**：
- 当 `y <= startY` 且 `flightTime > 0.1` 时触发爆炸
- 避免刚抛出就立即爆炸

#### 抛物线参数
