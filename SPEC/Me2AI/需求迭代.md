# 需求迭代

1. 先按照项目介绍文档初始化这个项目，跑出一个最小可运行的微信小程序；
2. 根据游戏介绍完成首页功能，其他功能先不实现；
3. 根据游戏介绍完成房间界面和功能，其他功能先不实现；
4. 根据游戏介绍完成选角色界面和功能，角色形象可以设计了，其他功能先不实现；
5. 开发角色系统，角色人物形象UI的设计，其他功能先不实现；
6. 选择角色的卡片优化成手势循环滚动的，以便适配后面增加角色；
7. 开发对战场景，实现角色移动、跳跃、攻击的控制逻辑，其他功能先不实现；
8. 重构角色系统，使得角色UI能生动形象，跑步要有跑步动作、举盾要有举盾动作，挥砍要有挥砍动作，挥剑要有挥剑动作，射击要有射击动作，跳跃要有跳跃动作，角色扣血时会从身上显示扣血值，被暴击时扣血效果加重；
