# 需求介绍

该项目是一个2.D横屏微信小程序游戏项目，类似于拳皇对战；

## 项目技术栈

Node22 + Pnpm8 + Typescript + cloudbase

cloudbase 云开发，对局即时通信等依赖服务器的功能都由它提供；

微信appId取config/secrets.js的APP_ID；
cloudbase env取config/secrets.js的CLOUD_BASE_ENV；

## 项目管理

采用 git + husky + lint-staged + commitizen 管理项目；
维护两个example文件，这些文件包含敏感信息，上传到git时需要忽略，example则是去掉敏感信息后的源文件；
维护一个 project.config.example.json，同步 project.config.json 的内容，但需将 appid 置空；
维护一个 src/config/secrets.example.ts，同步 src/config/secrets.ts 的内容，但需将所有字段值置空；

## 项目入口

以src作为工作区；
